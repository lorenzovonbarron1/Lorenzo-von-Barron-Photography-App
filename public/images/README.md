# Image Drop-In Guide

Drop real photography into these exact paths and the site picks them up
automatically. Until a file exists, the site renders a dark editorial
placeholder with the frame's label — nothing breaks.

Treatment notes: every image is rendered through a consistent dark veil and
gentle desaturation, so shoot/export slightly brighter than final. Hard-edged
crops, `object-fit: cover` — export generously sized (hero ≥ 2400px wide).
